
  # VITA Discovery Mobile Prototype

  This is a code bundle for VITA Discovery Mobile Prototype. The original project is available at https://www.figma.com/design/jXCjAgdgwKXsryxSFVNlDb/VITA-Discovery-Mobile-Prototype.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  Run `node index.js` from the `backend` folder to begin the database.

  ## Publishing the website

  Run `npm run build` to build the project.

  Once built, run `npm run deploy` to publish the build to vita-vu.github.io.
